# README #
### Папка offlineclient - инсталятор офлайн-клиента ###
Инсталятор оффлайн клиента устанавливает:
* дистрибутивы:
** java
** tomcat, регистрируется как сервис
** СУБД - по умолчанию - mysql, регистрирует как сервис
* Платформа и модель ION
** разворачивает платформу ion в томкат
** разворачивает модель офф-лайн клиента на платформу
** подгружает базу данных в СУБД
** производит настройки доступа к базе данных, настройки модели
* Сервис синхронизации
* разворачивает сервис синхронизации клиента
* записывает настройки для сервиса синхроинзации
